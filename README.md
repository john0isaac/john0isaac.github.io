# John Aziz Site

This repository contains the source for johnaziz.org. The site is generated by a small Python script using Jinja2 templates, Markdown content, and YAML-backed data.

## Getting Started

### Prerequisites

- Python 3.12+

### Installation

1. Clone the repository.
2. Create and activate a virtual environment.
3. Install dependencies:

```sh
uv sync
```

### Running Locally

Build the site (with minification and image optimization):

```sh
make build
```

Serve the site locally:

```sh
make serve
```

Or run via tox directly:

```sh
tox run -e build   # build
tox run -e serve   # serve
```

To skip minification or optimization during development:

```sh
python main.py build --no-minify --no-optimize
python main.py serve --no-minify --no-optimize
```

The generated output is written to `site/`.

## Content Sources

- `src/`: Markdown pages and blog posts.
- `data/`: YAML data for projects and talks.
- `assets/`: shared CSS and JavaScript used by the generated site.

## Output

The build generates static HTML plus `rss.xml`, `sitemap.xml`, `search.json`, `robots.txt`, and a 404 page.
